# Markdown Web

React + Vite + TypeScript app for editing Markdown with a live preview, multiple Markdown flavors (GitHub, CommonMark, Strict), file upload, and exports to `.md`, `.html`, and `.pdf`. UI is built with [shadcn/ui](https://ui.shadcn.com/) and Tailwind CSS. Theme follows the system by default, with optional light or dark mode.

## Scripts

```bash
npm install
npm run dev
```

Open the URL Vite prints (usually `http://localhost:5173`). Build for production with `npm run build` and preview the build with `npm run preview`.

## Lint

```bash
npm run lint
```
